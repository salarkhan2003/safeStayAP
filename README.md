# SafeStay AP

SafeStay AP is a secure platform designed in coordination with the AP Police Department for transparent, verified Paying Guest (PG) profiling and secure bookings.

## Key Features

- **Verified PGs**: Search and filter properties approved and secured by thorough background checks.
- **eKYC Integration**: Secure identity verification approved and monitored by the **AP Police Department** to guarantee authentic guest identities.
- **Multilingual Support**: Fully accessible interface with support for **English, Telugu, and Hindi** languages.
- **SOS Alert & Incident Reporting**: Ensures top-notch safety and fast response from local authorities.
- **Guest Pass**: A digital ID for quick and verified check-ins at your PG.

## Getting Started

1. Install all required dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npx expo start
   ```
